Quick controls

  • Right-click empty NFA canvas to add a state.
  • Right-click a state to mark start/final, rename, delete, or create a transition.
  • Drag states to reposition them. Drag empty canvas to pan. Use mouse wheel to zoom.
  • Use Define NFA to type the automaton directly, or load one of the presets.
  • Use epsilon or e for epsilon transitions. Use comma-separated symbols for grouped labels.

About Venom

VENOM stands for Virtual Engine for NFA to DFA Convertor and Visualiser.

String Acceptance

Leave it empty to test the empty string. Use comma-separated input for multi-character symbols.

Define NFA

Use one transition per line: from,symbol,to. Use eps or epsilon for epsilon transitions.
