Venom – NFA to DFA Visualizer 🚀
https://xharma-anurag.github.io/VENOM--/

Venom is an interactive Automata Theory project that helps users visualize and understand the conversion of Nondeterministic Finite Automata (NFA) into Deterministic Finite Automata (DFA) through an intuitive graphical interface.

The project is designed for students, educators, and developers who want a modern and dynamic way to explore automata concepts visually instead of relying only on theoretical explanations.

✨ Features
🔄 NFA to DFA Conversion
📊 Step-by-step conversion visualization
🎨 Interactive graph rendering
✅ Clear indication of start and final states
⚡ Dynamic and responsive UI
🧠 Supports custom automata input
📌 Transition mapping visualization
🌙 Modern dark-themed interface
🛠️ Tech Stack
HTML5
CSS3
JavaScript (ES6)
Graph Visualization Libraries
📚 Purpose

This project was built as an educational tool to simplify complex automata concepts and make learning finite automata more engaging and interactive.
